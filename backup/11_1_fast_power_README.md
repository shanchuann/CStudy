# 使用位运算实现快速幂算法

## 问题描述
实现一个使用位运算的快速幂算法，用于高效计算大数的幂运算。该算法应该能够处理正数、负数和零指数的情况。

## 要求
1. 实现以下功能：
   - 使用位运算进行快速幂计算
   - 处理正数、负数和零指数
   - 提供测试用例
   - 支持用户交互输入

2. 算法特点：
   - 时间复杂度为O(log n)
   - 使用位运算优化性能
   - 处理大数运算
   - 处理边界情况

## 算法原理
快速幂算法的核心思想是将指数表示为二进制形式，通过位运算来加速计算过程：
1. 将指数转换为二进制
2. 从低位到高位遍历每一位
3. 如果当前位为1，则将结果乘以底数
4. 底数不断平方
5. 指数右移一位

## 输入输出格式
### 输入
- 通过用户交互输入：
  - 底数（整数）
  - 指数（整数）

### 输出
- 显示计算结果
- 显示测试用例结果
