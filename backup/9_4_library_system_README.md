# 练习9.4：图书管理系统

## 题目描述
编写一个程序，实现图书管理系统。要求：
- 使用结构体数组存储图书信息
- 实现图书的增删改查功能
- 支持图书借阅、归还和预约功能
- 提供友好的用户界面

## 要求
1. 实现图书信息结构
   - 图书编号（整数）
   - 书名（字符串）
   - 作者（字符串）
   - 价格（浮点数）
   - 借阅状态（枚举）
   - 借阅者姓名（字符串）
   - 借阅日期（字符串）

2. 实现系统功能
   - 添加图书信息
   - 借阅图书
   - 归还图书
   - 预约图书
   - 显示所有图书

3. 实现数据管理
   - 使用结构体数组存储
   - 维护图书数量
   - 处理数组边界
   - 实现状态管理

4. 实现用户界面
   - 显示功能菜单
   - 处理用户输入
   - 显示操作结果
   - 提供错误提示

## 输入格式
- 通过菜单选择操作
- 输入图书信息
- 输入借阅信息
- 输入归还信息
- 输入预约信息

## 输出格式
- 显示操作菜单
- 显示图书信息列表
- 显示操作结果
- 使用表格形式展示数据

输出：

## 提示
1. 使用结构体定义图书信息
2. 使用枚举定义借阅状态
3. 实现基本的增删改查
4. 注意数据有效性检查
5. 优化用户交互体验
