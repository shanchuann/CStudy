# 练习6.9：实现数组的快速排序

## 题目描述
编写一个程序，使用快速排序算法对给定的整数数组进行排序。快速排序是一种高效的排序算法，通过选择一个基准值，将数组分成两部分，然后递归地对这两部分进行排序。

## 要求
1. 实现一个函数`quickSort`，接收数组和排序范围作为参数
2. 实现一个函数`partition`，用于选择基准值并分区
3. 实现一个函数`swap`，用于交换数组中的两个元素
4. 使用递归方式实现快速排序
5. 处理输入验证

## 输入格式
- 第一行输入数组大小n
- 第二行输入n个整数

## 输出格式
- 输出排序后的数组，每个数字之间用空格分隔

## 示例
输入：
```
6
64 34 25 12 22 11
```
输出：
```
排序后的数组: 11 12 22 25 34 64
```

## 提示
1. 选择最后一个元素作为基准值
2. 使用双指针技术进行分区
3. 注意递归的终止条件
4. 考虑使用动态内存分配
5. 可以添加排序过程的输出
