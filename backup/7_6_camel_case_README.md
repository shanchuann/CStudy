# 练习7.6：将字符串转换为驼峰命名法

## 题目描述
编写一个程序，将输入的字符串转换为驼峰命名法。驼峰命名法是一种命名规范，其中每个单词的首字母大写（除了第一个单词），其他字母小写，单词之间不使用空格或特殊字符。例如，"hello world"转换为"helloWorld"。

## 要求
1. 实现一个函数`toCamelCase`，接收字符串作为参数
2. 处理空格和特殊字符
3. 正确处理大小写
4. 保持第一个单词的首字母小写
5. 处理空字符串和NULL的情况

## 输入格式
- 输入一行包含空格或特殊字符的字符串

## 输出格式
- 输出转换后的驼峰命名法字符串

## 示例
输入：
```
hello world
```
输出：
```
驼峰命名法: helloWorld
```

## 提示
1. 使用双指针技术
2. 使用标记控制大小写
3. 跳过空格和特殊字符
4. 注意字符串结束符的处理
5. 考虑使用更安全的输入函数
