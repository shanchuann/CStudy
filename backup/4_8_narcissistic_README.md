# 练习4-8：找出水仙花数

## 题目描述
编写一个程序，找出所有的水仙花数。水仙花数是指一个三位数，其各位数字的立方和等于该数本身。

## 输出格式
按照以下格式输出水仙花数：
- 每个水仙花数占一行
- 格式为：n = a^3 + b^3 + c^3
- 其中n是水仙花数，a、b、c分别是其百位、十位、个位数字

## 示例输出
```
所有的水仙花数有：
153 = 1^3 + 5^3 + 3^3
370 = 3^3 + 7^3 + 0^3
371 = 3^3 + 7^3 + 1^3
407 = 4^3 + 0^3 + 7^3
```

## 提示
1. 遍历100到999的所有三位数
2. 分离数字的各位：
   - 百位：num / 100
   - 十位：(num / 10) % 10
   - 个位：num % 10
3. 计算各位数字的立方和
4. 判断是否等于原数
5. 按要求格式输出结果

## 学习要点
1. 数字分离
2. 循环结构
3. 条件判断
4. 格式化输出
