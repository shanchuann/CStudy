# 练习4-4：验证哥德巴赫猜想

## 题目描述
编写一个程序，验证哥德巴赫猜想：任何一个大于2的偶数都可以表示为两个素数之和。

## 输入格式
- 输入一个大于2的偶数

## 输出格式
- 输出该数可以表示为哪些素数对之和
- 每行输出一种表示方式：n = p1 + p2
- 如果输入无效，输出错误提示

## 示例
输入：
```
20
```
输出：
```
20可以表示为以下素数之和：
20 = 3 + 17
20 = 7 + 13
```

## 提示
1. 首先实现判断素数的函数isPrime
2. 输入验证：确保输入的是大于2的偶数
3. 遍历2到n/2的所有数，寻找素数对
4. 对于每个数i，检查i和n-i是否都是素数
5. 输出所有满足条件的素数对

## 学习要点
1. 函数定义和使用
2. 素数判断
3. 输入验证
4. 数学猜想验证
