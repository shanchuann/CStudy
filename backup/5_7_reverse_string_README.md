# 练习5-7：实现字符串反转函数（递归版）

## 题目描述
编写一个递归函数，将输入的字符串反转。函数应该能够处理任意长度的字符串。

## 函数要求
- 函数名：reverseString
- 参数：
  - char* str：要反转的字符串
  - int start：起始位置
  - int end：结束位置
- 返回值：void

## 输入格式
- 输入一个字符串（不包含空格）

## 输出格式
- 输出反转后的字符串

## 示例
输入：
```
hello
```
输出：
```
反转后的字符串：olleh
```

输入：
```
world
```
输出：
```
反转后的字符串：dlrow
```

## 提示
1. 函数设计：
   - 函数名：reverseString
   - 参数：
     - char* str：要反转的字符串
     - int start：起始位置
     - int end：结束位置
   - 返回值：void

2. 递归实现：
   - 基本情况：start >= end
   - 递归情况：
     - 交换首尾字符
     - 递归处理中间部分

3. 字符交换：
   - 使用临时变量
   - 注意字符位置

4. 主函数：
   - 输入字符串
   - 计算字符串长度
   - 调用反转函数
   - 输出结果

5. 注意事项：
   - 字符串边界检查
   - 递归深度控制
   - 内存安全

## 学习要点
1. 递归函数
2. 字符串操作
3. 字符交换
