1. 函数设计：
   - 函数名：stringMatch
   - 参数：
     - const char* text：文本串
     - const char* pattern：模式串
   - 返回值：int（匹配位置，-1表示未找到）

2. 特殊情况处理：
   - 模式串为空：返回0
   - 模式串长度大于文本串：返回-1
   - 未找到匹配：返回-1

3. 暴力匹配实现：
   - 外层循环遍历文本串
   - 内层循环比较字符
   - 找到匹配返回位置
   - 未找到继续搜索

4. 主函数：
   - 输入文本串和模式串
   - 调用匹配函数
   - 输出匹配结果

5. 注意事项：
   - 字符串边界检查
   - 数组大小限制
   - 返回值含义
   - 内存安全
