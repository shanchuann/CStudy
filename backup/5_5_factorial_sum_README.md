# 练习5-5：用函数重写阶乘和

## 题目描述
编写两个函数：一个计算阶乘，一个计算阶乘和。使用递归实现阶乘计算，使用循环实现阶乘和计算。

## 函数要求
1. 阶乘函数：
   - 函数名：factorial
   - 参数：int n
   - 返回值：long long
   - 使用递归实现

2. 阶乘和函数：
   - 函数名：factorialSum
   - 参数：int n
   - 返回值：long long
   - 使用循环实现

## 输入格式
- 输入一个非负整数n

## 输出格式
- 如果输入无效（负数），输出错误提示
- 如果输入有效，输出：1!+2!+...+n! = 结果

## 示例
输入：
```
5
```
输出：
```
1!+2!+...+5! = 153
```

输入：
```
-1
```
输出：
```
请输入非负整数！
```

## 提示
1. 阶乘函数设计：
   - 函数名：factorial
   - 参数：int n
   - 返回值：long long
   - 使用递归实现

2. 阶乘和函数设计：
   - 函数名：factorialSum
   - 参数：int n
   - 返回值：long long
   - 使用循环累加

3. 特殊情况处理：
   - n <= 1 时阶乘为1
   - n < 0 时输入无效

4. 数据类型选择：
   - 使用long long避免溢出
   - 注意n不要太大

5. 主函数：
   - 输入验证
   - 调用阶乘和函数
   - 输出结果

## 学习要点
1. 递归函数
2. 循环结构
3. 大数处理
4. 函数组合
