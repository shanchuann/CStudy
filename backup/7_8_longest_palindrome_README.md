# 练习7.8：计算最长回文子串长度

## 题目描述
编写一个程序，计算给定字符串中最长的回文子串的长度。回文串是指正着读和倒着读都一样的字符串。

例如：
- "babad" 中最长的回文子串是 "bab" 或 "aba"，长度为3
- "cbbd" 中最长的回文子串是 "bb"，长度为2
- "a" 的最长回文子串是 "a"，长度为1

## 要求
1. 实现一个函数`longestPalindrome`，接收一个字符串作为参数
2. 使用动态规划方法实现
3. 返回最长回文子串的长度
4. 处理空字符串的情况
5. 考虑字符串长度限制

## 输入格式
- 输入一个字符串

## 输出格式
- 输出最长回文子串的长度

## 示例
输入：
```
babad
```
输出：
```
最长回文子串的长度: 3
```

## 提示
1. 使用动态规划表记录子串是否为回文
2. 从短到长计算所有可能的子串
3. 注意处理边界情况
4. 考虑空间复杂度优化
5. 可以扩展输出最长回文子串本身
