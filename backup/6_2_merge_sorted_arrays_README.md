# 合并有序数组

## 题目描述
编写一个函数，将两个已排序的数组合并成一个新的有序数组。

## 函数要求
- 函数名：mergeSortedArrays
- 参数：
  - int arr1[]: 第一个有序数组
  - int size1: 第一个数组大小
  - int arr2[]: 第二个有序数组
  - int size2: 第二个数组大小
  - int result[]: 结果数组
- 返回值：void

## 输入格式
- 第一行输入第一个数组的大小n1
- 第二行输入n1个升序排列的整数
- 第三行输入第二个数组的大小n2
- 第四行输入n2个升序排列的整数

## 输出格式
- 输出合并后的有序数组

## 示例
输入：
```
3
1 3 5
3
2 4 6
```
输出：
```
合并后的数组：1 2 3 4 5 6
```
