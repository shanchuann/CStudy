# 练习9.6：实现时间结构体的加减操作

## 题目描述
编写一个程序，实现时间结构体的加减操作。要求：
- 使用结构体存储时间信息
- 实现时间的加减运算
- 处理时间进位和借位
- 验证时间有效性

## 要求
1. 实现时间结构体
   - 年（整数）
   - 月（整数）
   - 日（整数）
   - 时（整数）
   - 分（整数）
   - 秒（整数）

2. 实现时间验证
   - 检查日期有效性
   - 检查时间有效性
   - 处理闰年情况
   - 处理月份天数

3. 实现时间运算
   - 时间加法
   - 时间减法
   - 处理进位
   - 处理借位

4. 实现用户界面
   - 输入时间信息
   - 显示计算结果
   - 提供错误提示
   - 格式化输出

## 输入格式
- 输入年、月、日、时、分、秒
- 分别输入两个时间
- 使用整数输入

## 输出格式
- 显示输入的时间
- 显示加法结果
- 显示减法结果
- 使用标准时间格式

## 示例
输入：
```
请输入第一个时间：
请输入年：2024
请输入月：3
请输入日：15
请输入时：10
请输入分：30
请输入秒：45

请输入第二个时间：
请输入年：1
请输入月：2
请输入日：3
请输入时：4
请输入分：5
请输入秒：6
```

输出：
```
时间计算程序
============

计算结果：
----------------------------------------
时间1：2024-03-15 10:30:45
时间2：0001-02-03 04:05:06

时间1 + 时间2 = 2025-05-18 14:35:51
时间1 - 时间2 = 2023-01-12 06:25:39
```

## 提示
1. 注意时间进位规则
2. 处理月份天数变化
3. 考虑闰年情况
4. 验证输入有效性
5. 格式化输出结果
