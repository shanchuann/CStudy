# 练习6.1：找出数组中的第二大元素

## 题目描述
编写一个程序，找出给定数组中的第二大元素。如果数组元素不足两个，则输出提示信息。

## 要求
1. 实现一个函数`findSecondLargest`，接收一个整型数组和数组大小作为参数
2. 函数应返回数组中的第二大元素
3. 如果数组元素不足两个，返回-1并输出提示信息
4. 主函数中实现数组的输入和结果的输出

## 输入格式
- 第一行输入一个整数n，表示数组大小
- 第二行输入n个整数，表示数组元素

## 输出格式
- 如果找到第二大元素，输出"第二大的元素是: x"
- 如果数组元素不足，输出"数组元素不足"

## 示例
输入：
```
5
1 2 3 4 5
```
输出：
```
第二大的元素是: 4
```

## 提示
1. 注意处理数组元素不足的情况
2. 考虑数组中可能存在重复元素的情况
3. 可以使用两个变量分别记录最大值和第二大值
