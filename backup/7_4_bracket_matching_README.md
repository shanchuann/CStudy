# 练习7.4：检测括号匹配（栈的数组实现）

## 题目描述
编写一个程序，使用栈（数组实现）来检测字符串中的括号是否匹配。程序需要支持三种类型的括号：()、[]、{}。例如，字符串"{([])}"是匹配的，而"{(})"是不匹配的。

## 要求
1. 实现栈的基本操作（初始化、入栈、出栈）
2. 实现括号匹配检测函数
3. 支持三种类型的括号
4. 处理栈满和栈空的情况
5. 使用数组实现栈结构

## 输入格式
- 输入一行包含括号的字符串

## 输出格式
- 输出括号是否匹配的结果

## 示例
输入：
```
{([])}
```
输出：
```
括号匹配正确
```

## 提示
1. 使用数组实现栈结构
2. 使用栈来存储左括号
3. 遇到右括号时检查栈顶元素
4. 注意处理栈空的情况
5. 考虑使用更安全的输入函数
