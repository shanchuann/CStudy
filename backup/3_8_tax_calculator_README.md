# 个人所得税计算器

编写一个程序，计算个人所得税。

## 描述

个人所得税计算是工资发放中的重要环节。这个练习将帮助你理解税率计算和条件判断的应用。

## 任务

编写一个程序，实现以下功能：
1. 从用户输入获取月收入（税前）
2. 计算应纳税所得额（月收入 - 5000）
3. 根据税率表计算应缴税额
4. 计算税后收入

税率表：
- 不超过3000元：3%，速算扣除数0
- 3000-12000元：10%，速算扣除数210
- 12000-25000元：20%，速算扣除数1410
- 25000-35000元：25%，速算扣除数2660
- 35000-55000元：30%，速算扣除数4410
- 55000-80000元：35%，速算扣除数7160
- 超过80000元：45%，速算扣除数15160

要求：
- 输入月收入（整数）
- 输出格式为：
  ```
  税前收入：X元
  应纳税所得额：X元
  应缴税额：X元
  税后收入：X元
  ```

## 学习要点

1. 税率计算逻辑
2. 条件判断
3. 数学计算
4. 格式化输出
