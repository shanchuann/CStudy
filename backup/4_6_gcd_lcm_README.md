# 练习4-6：计算最大公约数和最小公倍数

## 题目描述
编写一个程序，计算两个正整数的最大公约数（GCD）和最小公倍数（LCM）。

## 输入格式
- 输入两个正整数，用空格分隔

## 输出格式
- 如果输入无效（不是正整数），输出错误提示
- 如果输入有效，输出：
  - 最大公约数
  - 最小公倍数

## 示例
输入：
```
12 18
```
输出：
```
最大公约数：6
最小公倍数：36
```

## 提示
1. 使用欧几里得算法计算最大公约数：
   - 当b不为0时，计算a%b
   - 将b赋值给a，将a%b赋值给b
   - 重复直到b为0
2. 最小公倍数计算公式：
   - lcm(a,b) = (a * b) / gcd(a,b)
3. 输入验证：确保输入的是正整数
4. 分别实现gcd和lcm函数
5. 注意整数除法的结果

## 学习要点
1. 函数定义和使用
2. 欧几里得算法
3. 输入验证
4. 数学计算
