# 计算某日是该年的第几天

编写一个程序，计算给定的日期是该年的第几天。

## 描述

在日常生活中，我们经常需要计算某个日期是一年中的第几天。这个练习将帮助你理解日期计算和闰年判断的逻辑。

## 任务

编写一个程序，实现以下功能：
1. 从用户输入获取年、月、日
2. 判断该年是否为闰年
3. 计算该日期是该年的第几天

要求：
- 输入格式：年 月 日（用空格分隔）
- 判断闰年的规则：
  - 能被4整除但不能被100整除的年份是闰年
  - 能被400整除的年份是闰年
- 输出格式为："X年X月X日是该年的第X天"

## 学习要点

1. 闰年判断逻辑
2. 日期计算
3. 条件判断
4. 数组使用（存储每月天数）
