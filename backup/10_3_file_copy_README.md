# 实现文件复制功能

## 问题描述
编写一个程序，实现文件复制功能。程序需要能够复制任意类型的文件，并显示复制进度。程序应该能够处理大文件，并提供复制过程的反馈。

## 要求
1. 实现以下功能：
   - 读取源文件
   - 写入目标文件
   - 显示复制进度
   - 支持任意类型的文件（二进制文件）

2. 程序特点：
   - 使用缓冲区提高效率
   - 显示文件大小和复制进度
   - 处理文件打开和关闭错误
   - 验证源文件和目标文件的有效性

## 输入输出格式
### 输入
- 通过命令行参数指定：
  - 源文件名
  - 目标文件名

### 输出
- 显示操作过程，包含：
  - 源文件大小
  - 实时复制进度
  - 操作状态（成功/失败）
  - 源文件和目标文件信息

## 示例
```
文件复制程序
============

开始复制文件...
源文件大小: 1024 字节
复制进度: 100% (1024/1024 字节)
文件复制完成！
源文件: input.txt
目标文件: output.txt
```

## 提示
1. 使用二进制模式打开文件
2. 使用较大的缓冲区提高效率
3. 注意文件指针的正确关闭
4. 处理各种错误情况
5. 验证输入参数的有效性
