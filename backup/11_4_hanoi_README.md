# 递归解决汉诺塔问题

## 问题描述
实现一个使用递归方法解决汉诺塔问题的程序。汉诺塔问题是一个经典的递归问题，要求将n个盘子从起始柱子移动到目标柱子，移动过程中需要遵循以下规则：
1. 一次只能移动一个盘子
2. 大盘子不能放在小盘子上面
3. 可以使用辅助柱子

## 要求
1. 实现以下功能：
   - 动态分配内存存储盘子状态
   - 递归实现移动算法
   - 显示每一步移动过程
   - 统计移动步数
   - 支持1-8个盘子

2. 程序特点：
   - 使用动态内存管理
   - 可视化显示状态
   - 完整的错误处理
   - 用户友好的界面

## 数据结构
1. 柱子结构体：
   ```c
   typedef struct {
       int* disks;     // 盘子数组
       int size;       // 当前盘子数量
       int capacity;   // 最大容量
   } Tower;
   ```

## 主要功能
1. 内存管理：
   - 初始化柱子
   - 释放内存
   - 动态数组操作

2. 盘子操作：
   - 添加盘子
   - 移除盘子
   - 查看顶部盘子
   - 移动盘子

3. 显示功能：
   - 显示当前状态
   - 显示移动过程
   - 显示统计信息

## 输入输出格式
### 输入
- 通过用户输入指定盘子数量（1-8）

### 输出
- 显示初始状态
- 显示每一步移动后的状态
- 显示最终移动步数

## 示例
```
请输入盘子数量（1-8）：3

初始状态：
----------------------------------------
3        |       |
2        |       |
1        |       |
A        B       C
----------------------------------------

当前状态：
----------------------------------------
|       |       3
|       |       2
|       |       1
A       B       C
----------------------------------------

完成！总共移动 7 步。
```
