# 练习3-10：判断坐标点所在象限

## 题目描述
编写一个程序，根据用户输入的x和y坐标，判断该点位于哪个象限或坐标轴上。

## 输入格式
- 第一行输入x坐标（浮点数）
- 第二行输入y坐标（浮点数）

## 输出格式
根据坐标点位置输出以下信息之一：
- "该点位于第一象限"
- "该点位于第二象限"
- "该点位于第三象限"
- "该点位于第四象限"
- "该点位于坐标轴上"

## 示例
输入：
```
3.5
2.8
```
输出：
```
该点位于第一象限
```

## 提示
1. 使用float类型存储x和y坐标
2. 首先判断点是否在坐标轴上（x=0或y=0）
3. 使用if-else if-else结构判断象限
4. 确保输出格式与要求完全一致

## 学习要点

1. 坐标系统
2. 条件判断
3. 浮点数比较
4. 逻辑运算
