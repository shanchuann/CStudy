# 练习4-7：统计正负数个数

## 题目描述
编写一个程序，统计用户输入的一系列整数中正数和负数的个数，输入0时结束。

## 输入格式
- 输入一系列整数，用空格分隔
- 输入0表示结束

## 输出格式
- 输出正数个数和负数个数
- 格式为：
  - "正数个数：x"
  - "负数个数：y"

## 示例
输入：
```
1 -2 3 -4 5 0
```
输出：
```
正数个数：3
负数个数：2
```

## 提示
1. 使用while(1)循环实现无限输入
2. 使用break语句在输入0时退出循环
3. 使用两个计数器分别统计正数和负数
4. 使用if-else结构判断正负
5. 最后输出统计结果

## 学习要点
1. 循环控制
2. 条件判断
3. 计数器使用
4. 输入处理
