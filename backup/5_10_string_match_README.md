# 练习5-10：实现字符串匹配函数

## 题目描述
编写一个函数，实现字符串匹配功能。函数应该能够在文本串中查找模式串，并返回第一次匹配的位置。

## 函数要求
- 函数名：stringMatch
- 参数：
  - const char* text：文本串
  - const char* pattern：模式串
- 返回值：int
  - 如果找到匹配，返回第一次匹配的位置（从0开始）
  - 如果未找到匹配，返回-1
  - 如果模式串为空，返回0

## 输入格式
- 第一行输入文本串
- 第二行输入模式串

## 输出格式
- 如果找到匹配，输出匹配位置
- 如果未找到匹配，输出未找到提示

## 示例
输入：
```
hello
ll
```
输出：
```
模式串在位置 2 处匹配成功！
```

输入：
```
world
abc
```
输出：
```
未找到匹配！
```

## 提示
1. 函数设计：
   - 函数名：stringMatch
   - 参数：
     - const char* text：文本串
     - const char* pattern：模式串
   - 返回值：int（匹配位置，-1表示未找到）

2. 特殊情况处理：
   - 模式串为空：返回0
   - 模式串长度大于文本串：返回-1
   - 未找到匹配：返回-1

3. 暴力匹配实现：
   - 外层循环遍历文本串
   - 内层循环比较字符
   - 找到匹配返回位置
   - 未找到继续搜索

4. 主函数：
   - 输入文本串和模式串
   - 调用匹配函数
   - 输出匹配结果

5. 注意事项：
   - 字符串边界检查
   - 数组大小限制
   - 返回值含义
   - 内存安全

## 学习要点
1. 字符串操作
2. 暴力匹配算法
3. 特殊情况处理
4. 函数返回值设计
