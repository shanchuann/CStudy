1. 通用冒泡排序函数设计：
   - 函数名：bubbleSort
   - 参数：
     - void* arr：数组指针
     - int n：数组长度
     - int size：元素大小
     - int (*compare)(const void*, const void*)：比较函数指针
   - 返回值：void

2. 比较函数设计：
   - 整数比较：直接相减
   - 浮点数比较：使用差值判断
   - 返回值：>0表示a>b，<0表示a<b，=0表示a=b

3. 元素交换实现：
   - 使用临时缓冲区
   - 使用memcpy函数
   - 注意内存对齐
