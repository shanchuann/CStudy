# 找出三个整数中的最大值

编写一个程序，使用嵌套if语句找出三个整数中的最大值。

## 描述

在实际编程中，我们经常需要比较多个数值并找出其中的最大值。这个练习将帮助你理解嵌套if语句的使用方法，以及如何通过多重比较来解决问题。

## 任务

编写一个程序，实现以下功能：
1. 从用户输入获取三个整数
2. 使用嵌套if语句比较这三个数
3. 输出最大值

要求：
- 使用嵌套if语句实现比较逻辑
- 不能使用数组或循环
- 不能使用内置的最大值函数
- 输出格式为："最大值是：X"

## 学习要点

1. 嵌套if语句的使用
2. 多重条件判断的逻辑
3. 变量比较和交换
4. 程序流程控制
