# 练习5-3：编写函数求整数的二进制形式

## 题目描述
编写一个递归函数，将一个整数转换为二进制形式并打印。函数应该能够处理正数、负数和零。

## 函数要求
- 函数名：printBinary
- 参数：int n（要转换的整数）
- 返回值：void（直接打印结果）

## 输入格式
- 输入一个整数（可以是正数、负数或零）

## 输出格式
- 输出该整数的二进制形式
- 对于负数，先输出负号，再输出二进制形式
- 数字之间不需要空格

## 示例
输入：
```
42
```
输出：
```
42的二进制形式是：101010
```

输入：
```
-42
```
输出：
```
-42的二进制形式是：-101010
```

## 提示
1. 函数设计：
   - 函数名：printBinary
   - 参数：int n（要转换的整数）
   - 返回值：void（直接打印结果）

2. 特殊情况处理：
   - 负数：先打印负号，再处理绝对值
   - 零：直接打印0
   - 正数：递归处理

3. 递归实现：
   - 基本情况：n == 0
   - 递归情况：先递归处理n/2，再打印n%2

4. 主函数：
   - 输入整数
   - 调用printBinary函数
   - 输出结果

5. 注意事项：
   - 递归顺序要正确（先递归后打印）
   - 处理负数的特殊情况
   - 确保输出格式正确

## 学习要点
1. 递归函数
2. 进制转换
3. 特殊情况处理
4. 格式化输出
