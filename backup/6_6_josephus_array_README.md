# 练习6.6：约瑟夫环问题（数组实现）

## 题目描述
约瑟夫环问题是一个经典的数学问题。n个人围成一圈，从第一个人开始报数，报到k的人出列，然后从出列的下一个人开始重新报数，直到最后只剩下一个人。编写一个程序，使用数组实现约瑟夫环问题，找出最后存活的人的位置。

## 要求
1. 实现一个函数`josephus`，接收总人数n和报数k作为参数
2. 使用数组模拟约瑟夫环
3. 返回最后存活的人的位置（从1开始计数）
4. 处理输入验证

## 输入格式
- 第一行输入总人数n
- 第二行输入报数k

## 输出格式
- 输出最后存活的人的位置

## 示例
输入：
```
5
2
```
输出：
```
最后存活的人的位置是: 3
```

## 提示
1. 使用数组表示每个人（1表示存活，0表示被淘汰）
2. 使用循环模拟报数过程
3. 注意数组索引的处理
4. 考虑使用动态内存分配处理大n
5. 可以添加过程输出
