# 练习11.6：用宏实现泛型交换函数

## 题目描述
使用C语言宏实现一个通用的交换函数，能够交换任意类型的数据。

## 要求
1. 实现一个通用的SWAP宏
2. 支持基本数据类型（int, float, double, char等）
3. 支持结构体类型
4. 确保类型安全
5. 避免临时变量带来的内存开销

## 示例代码框架
```c
// 基本实现
#define SWAP(x, y) do { \
    typeof(x) temp = x; \
    x = y;             \
    y = temp;          \
} while(0)

// 使用示例
int a = 10, b = 20;
SWAP(a, b);

float f1 = 1.5f, f2 = 2.5f;
SWAP(f1, f2);

struct Point p1 = {1, 2}, p2 = {3, 4};
SWAP(p1, p2);
```

## 提示
1. 使用typeof运算符获取类型信息
2. 考虑内存对齐问题
3. 注意宏展开可能带来的副作用
4. 处理大对象的交换效率

## 扩展要求

2. 添加类型检查机制
3. 实现数组元素的交换
