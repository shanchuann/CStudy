# 实现文件内容加密（异或运算）

## 问题描述
编写一个程序，使用异或运算对文件内容进行加密和解密。程序需要能够处理任意类型的文件，并使用用户指定的密钥进行加密/解密操作。

## 要求
1. 实现以下功能：
   - 读取输入文件
   - 使用异或运算加密/解密文件内容
   - 将结果写入输出文件
   - 支持任意类型的文件（二进制文件）

2. 程序特点：
   - 使用缓冲区提高效率
   - 处理文件打开和关闭错误
   - 验证密钥的有效性
   - 提供操作结果反馈

## 输入输出格式
### 输入
- 通过命令行参数指定：
  - 输入文件名
  - 输出文件名
  - 加密密钥（0-255之间的整数）

### 输出
- 显示操作结果，包含：
  - 操作状态（成功/失败）
  - 输入文件信息
  - 输出文件信息
  - 使用的密钥

## 示例
```
文件加密/解密程序
===============

文件加密/解密成功！
输入文件: input.txt
输出文件: output.enc
使用密钥: 123
```

## 提示
1. 使用二进制模式打开文件
2. 使用缓冲区读取大文件
3. 注意文件指针的正确关闭
4. 处理各种错误情况
5. 验证输入参数的有效性
