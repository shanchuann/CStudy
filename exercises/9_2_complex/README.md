# 练习9.2：实现复数运算

## 题目描述
编写一个程序，实现复数的基本运算。要求：
- 使用结构体表示复数
- 实现复数的加减乘除运算
- 计算复数的模和共轭
- 提供友好的用户界面

## 要求
1. 实现复数结构
   - 实部（double类型）
   - 虚部（double类型）
   - 创建和打印函数

2. 实现基本运算
   - 复数加法
   - 复数减法
   - 复数乘法
   - 复数除法
   - 计算模
   - 计算共轭

3. 实现错误处理
   - 处理除数为零
   - 处理无效输入
   - 处理精度问题

4. 实现用户界面
   - 显示功能菜单
   - 处理用户输入
   - 显示计算结果
   - 提供错误提示

## 输入格式
- 通过菜单选择操作
- 输入复数实部和虚部
- 输入运算参数
- 输入计算选项

## 输出格式
- 显示操作菜单
- 显示复数表达式
- 显示计算结果
- 使用标准数学表示

## 示例
输出：
```
复数运算系统
1. 复数加法
2. 复数减法
3. 复数乘法
4. 复数除法
5. 计算模
6. 计算共轭
0. 退出系统
请选择操作：1

请输入第一个复数：
请输入实部：3
请输入虚部：4
请输入第二个复数：
请输入实部：1
请输入虚部：2
(3.00 + 4.00i) + (1.00 + 2.00i) = 4.00 + 6.00i
```

1. 使用结构体定义复数
2. 实现基本运算函数

4. 处理特殊情况
5. 优化输出格式
