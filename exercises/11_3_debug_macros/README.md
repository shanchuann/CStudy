# 使用预处理器实现调试宏

## 问题描述
实现一组功能强大的调试宏，用于程序开发和调试过程中输出调试信息、进行断言检查、跟踪函数调用和内存操作等。

## 要求
1. 实现以下功能：
   - 不同级别的调试信息输出
   - 断言检查
   - 内存分配检查
   - 函数调用跟踪
   - 时间戳记录
   - 文件行号记录

2. 调试宏特点：
   - 支持多个调试级别
   - 包含时间戳信息
   - 包含文件名和行号
   - 支持格式化输出
   - 条件编译支持

## 调试级别
1. 错误级别（ERROR）：
   - 用于输出错误信息
   - 最高优先级
   - 始终显示

2. 警告级别（WARN）：
   - 用于输出警告信息
   - 次高优先级
   - 在警告级别及以上显示

3. 信息级别（INFO）：
   - 用于输出一般信息
   - 中等优先级
   - 在信息级别及以上显示

4. 调试级别（DEBUG）：
   - 用于输出详细调试信息
   - 最低优先级
   - 仅在调试级别显示

## 主要宏定义
1. 调试输出宏：
   ```c
   #define DEBUG_ERROR(fmt, ...)
   #define DEBUG_WARN(fmt, ...)
   #define DEBUG_INFO(fmt, ...)
   #define DEBUG_DEBUG(fmt, ...)
   ```

2. 断言宏：
   ```c
   #define ASSERT(condition)
   ```

3. 内存检查宏：
   ```c
   #define SAFE_MALLOC(ptr, size)
   ```

4. 函数跟踪宏：
   ```c
   #define FUNCTION_ENTRY()
   #define FUNCTION_EXIT()
   ```

## 输出格式
调试信息输出格式：
```
[时间戳][文件名:行号] 级别: 消息内容
```

示例：
```
[2024-01-01 10:00:00][main.c:42] ERROR: 内存分配失败
[2024-01-01 10:00:00][main.c:45] WARN: 接收到负值
[2024-01-01 10:00:00][main.c:48] INFO: 处理数值: 42
[2024-01-01 10:00:00][main.c:50] DEBUG: 进入函数: exampleFunction
```

## 提示
1. 使用do-while(0)包装宏定义
2. 使用__FILE__和__LINE__获取位置信息
3. 使用__FUNCTION__获取函数名
4. 使用可变参数宏
5. 注意宏展开的安全性
