# 练习9.1：学生成绩管理系统

## 题目描述
编写一个程序，实现学生成绩管理系统。要求：
- 使用结构体数组存储学生信息
- 实现学生信息的增删改查功能
- 支持按平均分排序
- 提供友好的用户界面

## 要求
1. 实现学生信息结构
   - 学号（整数）
   - 姓名（字符串）
   - 语文成绩（浮点数）
   - 数学成绩（浮点数）
   - 英语成绩（浮点数）
   - 平均分（浮点数）

2. 实现系统功能
   - 添加学生信息
   - 修改学生信息
   - 删除学生信息
   - 显示所有学生
   - 按平均分排序

3. 实现数据管理
   - 使用结构体数组存储
   - 维护学生数量
   - 处理数组边界
   - 实现数据排序

4. 实现用户界面
   - 显示功能菜单
   - 处理用户输入
   - 显示操作结果
   - 提供错误提示

## 输入格式
- 通过菜单选择操作
- 输入学生信息
- 输入修改数据
- 输入删除条件

## 输出格式
- 显示操作菜单
- 显示学生信息列表
- 显示操作结果
- 使用表格形式展示数据

输出：

## 提示
1. 使用结构体定义学生信息
2. 使用数组管理多个学生
3. 实现基本的增删改查
4. 注意数据有效性检查
5. 优化用户交互体验
