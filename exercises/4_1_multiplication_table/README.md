# 练习4-1：打印乘法口诀表

## 题目描述
编写一个程序，打印九九乘法口诀表，要求格式对齐。

## 输出格式
按照以下格式输出乘法口诀表：
- 每行从1开始，到当前行数结束
- 每个算式后保留3个字符的宽度
- 每行结束后换行

## 示例输出
```
1×1=1  
1×2=2  2×2=4  
1×3=3  2×3=6  3×3=9  
...
```

## 提示
1. 使用两个嵌套的for循环实现
2. 外层循环控制行数（1-9）
3. 内层循环控制每行的列数（1到当前行数）
4. 使用printf的格式控制符%-3d确保对齐
5. 每行结束后需要换行

## 学习要点
1. 嵌套循环的使用
2. 格式化输出
3. 程序结构组织
