# 练习7.7：实现简单的正则表达式匹配（仅支持.和*）

## 题目描述
编写一个程序，实现简单的正则表达式匹配功能。支持两种特殊字符：
- '.' 匹配任意单个字符
- '*' 匹配零个或多个前面的字符

例如：
- "a*" 可以匹配 "", "a", "aa", "aaa" 等
- ".*" 可以匹配任意字符串
- "a.b" 可以匹配 "aab", "acb", "adb" 等

## 要求
1. 实现一个函数`isMatch`，接收字符串和模式串作为参数
2. 支持'.'和'*'两种特殊字符
3. 使用递归方式实现
4. 处理空字符串的情况
5. 正确处理模式串的边界情况

## 输入格式
- 第一行输入要匹配的字符串
- 第二行输入模式串

## 输出格式
- 输出匹配结果（成功/失败）

## 示例
输入：
```
aa
a*
```
输出：
```
匹配成功
```

## 提示
1. 使用递归实现匹配
2. 分别处理*和非*的情况
3. 注意处理空字符串
4. 考虑使用动态规划优化
5. 可以添加更多特殊字符支持
