# 练习6.7：找出数组中重复出现过的元素

## 题目描述
编写一个程序，找出给定数组中所有重复出现的元素。如果一个元素在数组中出现了两次或更多次，就认为它是重复的。程序需要输出所有重复的元素。

## 要求
1. 实现一个函数`findDuplicates`，接收一个整型数组和数组大小作为参数
2. 使用标记数组记录元素出现次数
3. 输出所有重复出现的元素
4. 处理没有重复元素的情况

## 输入格式
- 第一行输入数组大小n
- 第二行输入n个整数

## 输出格式
- 如果有重复元素，输出"重复的元素是: "后跟所有重复元素
- 如果没有重复元素，输出"没有找到重复的元素"

## 示例
输入：
```
6
1 2 3 2 4 3
```
输出：
```
重复的元素是: 2 3
```

## 提示
1. 使用标记数组记录元素出现次数
2. 注意数组大小限制
3. 考虑使用动态内存分配
4. 可以优化输出格式
5. 考虑使用排序方法
