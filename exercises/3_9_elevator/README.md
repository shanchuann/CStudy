# 电梯楼层选择逻辑

编写一个程序，模拟电梯的楼层选择逻辑。

## 描述

电梯是日常生活中常见的设备，它需要根据用户的选择和当前状态来决定运行方向。这个练习将帮助你理解电梯控制逻辑和状态管理。

## 任务

编写一个程序，实现以下功能：
1. 从用户输入获取当前楼层和目标楼层
2. 根据当前楼层和目标楼层判断电梯运行方向
3. 输出电梯运行状态

要求：
- 假设楼层范围为1-20层
- 电梯运行规则：
  - 如果目标楼层大于当前楼层，电梯向上运行
  - 如果目标楼层小于当前楼层，电梯向下运行
  - 如果目标楼层等于当前楼层，电梯停止
- 输出格式为：
  ```
  当前楼层：X
  目标楼层：X
  电梯状态：向上运行/向下运行/停止
  ```

## 学习要点

1. 状态判断
2. 条件分支
3. 输入验证
4. 逻辑控制
