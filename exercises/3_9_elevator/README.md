电梯调度系统

编写一个程序，模拟电梯的运行和调度。
这个练习将帮助你理解状态机、条件判断和循环控制。

任务
编写一个程序，实现以下功能：
1. 从用户输入获取目标楼层（1-20层）
2. 根据当前楼层和目标楼层确定运行方向
3. 模拟电梯运行过程
4. 输出当前楼层
- 输入必须为有效楼层（1-20）
- 电梯运行方向：
  * 上行：目标楼层 > 当前楼层
  * 下行：目标楼层 < 当前楼层
  * 停止：目标楼层 = 当前楼层
- 输出格式为："Current floor: X" 或 "Invalid floor"

学习要点
1. 状态机的实现
2. 条件判断和循环控制
3. 数据有效性检查
4. 程序优化
