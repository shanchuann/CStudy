# 练习5-4：实现冒泡排序的通用函数

## 题目描述
编写一个通用的冒泡排序函数，可以对任意类型的数组进行排序。函数使用函数指针实现比较操作，支持整数、浮点数等不同类型。

## 函数要求
- 函数名：bubbleSort
- 参数：
  - void* arr：数组指针
  - int n：数组长度
  - int size：元素大小
  - int (*compare)(const void*, const void*)：比较函数指针
- 返回值：void

## 比较函数要求
- 整数比较函数：compareInt
- 浮点数比较函数：compareFloat
- 返回值：
  - >0：第一个元素大于第二个元素
  - <0：第一个元素小于第二个元素
  - =0：两个元素相等

## 示例
输入：
```
整数数组：[64, 34, 25, 12, 22, 11, 90]
浮点数数组：[3.14, 2.71, 1.41, 0.58, 1.73]
```
输出：
```
排序前的整数数组：
64 34 25 12 22 11 90
排序后的整数数组：
11 12 22 25 34 64 90

排序前的浮点数数组：
3.14 2.71 1.41 0.58 1.73
排序后的浮点数数组：
0.58 1.41 1.73 2.71 3.14
```

## 提示
1. 通用冒泡排序函数设计：
   - 函数名：bubbleSort
   - 参数：
     - void* arr：数组指针
     - int n：数组长度
     - int size：元素大小
     - int (*compare)(const void*, const void*)：比较函数指针
   - 返回值：void

2. 比较函数设计：
   - 整数比较：直接相减
   - 浮点数比较：使用差值判断
   - 返回值：>0表示a>b，<0表示a<b，=0表示a=b

3. 元素交换实现：
   - 使用临时缓冲区
   - 使用memcpy函数
   - 注意内存对齐

4. 主函数：
   - 测试整数数组
   - 测试浮点数数组
   - 打印排序前后结果

5. 注意事项：
   - 指针类型转换
   - 内存操作安全
   - 比较函数实现
