# 实现文件分割功能

## 问题描述
编写一个程序，实现文件分割功能。程序需要能够将一个大文件分割成多个小文件，每个分割文件的大小由用户指定。程序应该能够处理任意类型的文件，并提供分割过程的反馈。

## 要求
1. 实现以下功能：
   - 读取输入文件
   - 按指定大小分割文件
   - 生成分割文件
   - 显示分割进度
   - 支持任意类型的文件（二进制文件）

2. 程序特点：
   - 使用缓冲区提高效率
   - 显示文件大小和分割信息
   - 处理文件打开和关闭错误
   - 验证输入参数的有效性
   - 限制最大分割数量

## 输入输出格式
### 输入
- 通过命令行参数指定：
  - 输入文件名
  - 每部分大小（字节）

### 输出
- 显示操作过程，包含：
  - 文件大小信息
  - 分割数量
  - 每个分割文件的进度
  - 操作状态（成功/失败）
  - 分割结果统计

## 示例
```
文件分割程序
============

开始分割文件...
文件大小: 1024 字节
每部分大小: 256 字节
分割数量: 4

正在创建分割文件 1/4: input.txt.part1
文件 1 进度: 100% (256/256 字节)
正在创建分割文件 2/4: input.txt.part2
文件 2 进度: 100% (256/256 字节)
正在创建分割文件 3/4: input.txt.part3
文件 3 进度: 100% (256/256 字节)
正在创建分割文件 4/4: input.txt.part4
文件 4 进度: 100% (256/256 字节)

文件分割完成！
总字节数: 1024
分割文件列表:
  1. input.txt.part1
  2. input.txt.part2
  3. input.txt.part3
  4. input.txt.part4
```

## 提示
1. 使用二进制模式打开文件
2. 使用较大的缓冲区提高效率
3. 注意文件指针的正确关闭
4. 处理各种错误情况
5. 验证输入参数的有效性
