# 练习4-3：找出完数

## 题目描述
编写一个程序，找出1000以内的所有完数。完数是指一个数恰好等于它的所有真因子（不包括该数本身）之和。

## 输出格式
按照以下格式输出完数：
- 每个完数占一行
- 格式为：完数 = 因子1 + 因子2 + ... + 因子n
- 最后一个因子后不要输出加号

## 示例输出
```
1000以内的完数有：
6 = 1 + 2 + 3
28 = 1 + 2 + 4 + 7 + 14
496 = 1 + 2 + 4 + 8 + 16 + 31 + 62 + 124 + 248
```

## 提示
1. 使用两层循环实现：
   - 外层循环遍历1-1000的数
   - 内层循环找出当前数的所有真因子
2. 真因子的判断：能被该数整除且小于该数的数
3. 完数的判断：所有真因子之和等于原数
4. 输出格式：完数 = 因子1 + 因子2 + ...
5. 注意最后一个因子后不要输出加号

## 学习要点
1. 循环嵌套
2. 因子判断
3. 格式化输出
4. 数学概念应用
