# 设计交通信号灯状态机

## 问题描述
使用枚举和结构体实现一个交通信号灯状态机，模拟交通信号灯的工作过程。信号灯有三种状态：红灯、绿灯和黄灯，每种状态都有其持续时间。

## 要求
1. 定义信号灯状态枚举，包含三种状态
2. 定义信号灯结构体，包含：
   - 当前状态
   - 当前状态持续时间
   - 各状态持续时间设置
3. 实现以下功能：
   - 初始化信号灯
   - 更新信号灯状态
   - 获取当前状态
   - 设置各状态持续时间
   - 显示信号灯信息

## 输入输出格式
### 输入
- 初始化时输入三个整数值，分别表示红灯、绿灯、黄灯的持续时间（秒）
- 更新状态时不需要输入

### 输出
- 显示信号灯信息时，包含：
  - 当前状态
  - 当前状态持续时间
  - 各状态持续时间设置
- 格式示例：
  ```
  当前状态: 红灯
  持续时间: 25秒
  红灯时长: 30秒
  绿灯时长: 25秒
  黄灯时长: 5秒
  ```

## 示例
```
交通信号灯状态机程序
==================

初始状态:

持续时间: 0秒
红灯时长: 30秒
绿灯时长: 25秒
黄灯时长: 5秒
------------------------

第1次更新:
当前状态: 红灯
持续时间: 1秒
红灯时长: 30秒
绿灯时长: 25秒
黄灯时长: 5秒
------------------------
```

## 提示
1. 使用枚举类型定义信号灯状态
2. 使用结构体组织信号灯数据
3. 注意状态转换的逻辑
4. 考虑边界条件处理
5. 可以使用switch语句处理状态转换
