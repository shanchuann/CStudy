# 练习6.4：统计字符串中各元音字母出现次数

## 题目描述
编写一个程序，统计给定字符串中各个元音字母（a、e、i、o、u）出现的次数。程序应该不区分大小写，并输出每个元音字母的出现次数。

## 要求
1. 实现一个函数`countVowels`，接收一个字符串和计数数组作为参数
2. 使用数组存储各个元音字母的计数
3. 不区分大小写进行统计
4. 处理输入字符串中的特殊字符和空格

## 输入格式
- 输入一个字符串，可以包含任意字符

## 输出格式
- 按照以下格式输出每个元音字母的出现次数：
  ```
  元音字母统计结果：
  a: x
  e: x
  i: x
  o: x
  u: x
  ```

## 示例
输入：
```
Hello World!
```
输出：
```
元音字母统计结果：
a: 0
e: 1
i: 0
o: 2
u: 0
```

## 提示
1. 使用数组存储计数结果
2. 使用tolower()函数处理大小写
3. 注意字符串的输入和结束符处理
4. 考虑使用const修饰输入参数
5. 可以添加输入验证
