# 练习7.2：统计单词数量（考虑多个空格情况）

## 题目描述
编写一个程序，统计输入文本中的单词数量。单词是由字母或数字组成的连续字符序列，单词之间可能有一个或多个空格或其他分隔符。

## 要求
1. 实现一个函数`countWords`，接收字符串作为参数
2. 正确处理多个连续空格的情况
3. 将字母和数字视为单词的一部分
4. 处理输入验证
5. 使用标准库函数判断字符类型

## 输入格式
- 输入一行文本，可能包含多个空格

## 输出格式
- 输出统计得到的单词数量

## 示例
输入：
```
Hello   World!  This is a   test.
```
输出：
```
单词数量: 6
```

## 提示
1. 使用状态标记判断是否在单词内
2. 使用isalnum函数判断字符类型
3. 注意处理字符串开头和结尾的空格
4. 考虑使用更安全的输入函数
5. 可以添加单词长度统计
