# 练习8.7：实现简单的内存池

## 题目描述
编写一个程序，实现一个简单的内存池。要求：
- 使用固定大小的内存块
- 实现内存分配和释放功能
- 跟踪内存使用状态
- 提供内存池状态查询功能

## 要求
1. 实现内存池结构
   - 定义内存块结构
   - 定义内存池结构
   - 实现内存池初始化

2. 实现内存管理功能
   - 内存分配函数
   - 内存释放函数
   - 内存池销毁函数

3. 实现状态管理
   - 跟踪内存使用情况
   - 计算内存使用率
   - 显示内存池状态

4. 实现错误处理
   - 处理内存分配失败
   - 处理无效指针
   - 处理内存溢出

## 输入格式
- 程序内部提供测试数据
- 使用预定义的内存块大小
- 包含错误处理测试

## 输出格式
- 显示内存池创建过程
- 显示内存分配和释放信息
- 显示内存池状态信息
- 使用适当的格式化输出

## 示例
输出：
```
创建内存池...

内存池状态：
总大小：10240 字节
已使用：0 字节
使用率：0.00%

内存块状态：
块 0: 空闲 (大小: 1024 字节)
块 1: 空闲 (大小: 1024 字节)
...

分配内存...
分配的内存内容：Hello, Memory Pool!

内存池状态：
总大小：10240 字节
已使用：100 字节
使用率：0.98%
...
```

## 提示
1. 使用结构体管理内存块
2. 实现内存分配算法
3. 注意内存释放顺序
4. 处理边界情况
5. 优化内存使用效率
