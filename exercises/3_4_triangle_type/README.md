# 判断三角形类型

编写一个程序，根据三条边的长度判断三角形的类型。

## 描述

在几何学中，根据三角形的边长关系，可以将三角形分为等边三角形、等腰三角形、直角三角形和普通三角形。这个练习将帮助你理解三角形的性质和条件判断的应用。

## 任务

编写一个程序，实现以下功能：
1. 从用户输入获取三条边的长度
2. 判断这三条边是否能构成三角形
3. 如果能构成三角形，判断其类型：
   - 等边三角形：三条边相等
   - 等腰三角形：两条边相等
   - 直角三角形：满足勾股定理（a² + b² = c²）
   - 普通三角形：其他情况

要求：
- 输入三条边的长度（浮点数）
- 判断是否能构成三角形（任意两边之和大于第三边）
- 输出格式为："这是一个X三角形"
- 如果不能构成三角形，输出："这三条边不能构成三角形"

## 学习要点

1. 三角形的基本性质
2. 浮点数比较
3. 多重条件判断
4. 数学计算和公式实现
