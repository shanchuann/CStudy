# 练习4-9：猜数字游戏

## 题目描述
编写一个程序，实现一个简单的猜数字游戏。程序随机生成一个1-100之间的数字，用户有7次机会猜对这个数字。每次猜测后，程序会提示数字是太大还是太小。

## 输入格式
- 每次输入一个整数作为猜测

## 输出格式
- 游戏开始时显示欢迎信息和规则
- 每次猜测后显示提示信息（太大/太小）
- 猜对时显示祝贺信息和所用次数
- 用完所有机会后显示正确答案

## 示例
```
欢迎来到猜数字游戏！
我已经想好了一个1-100之间的数字，你有7次机会猜对它。

请输入你的猜测（还剩7次机会）：50
太大了！再小一点。

请输入你的猜测（还剩6次机会）：25
太小了！再大一点。

请输入你的猜测（还剩5次机会）：37
太小了！再大一点。

请输入你的猜测（还剩4次机会）：43
恭喜你猜对了！答案就是43。
你用了4次就猜到了！
```

## 提示
1. 使用srand和rand函数生成随机数
2. 使用time函数初始化随机数种子
3. 使用while循环控制猜测次数
4. 使用if-else结构判断猜测结果
5. 记录并显示剩余猜测次数

## 学习要点
1. 随机数生成
2. 循环控制
3. 条件判断
4. 用户交互
