# 实现颜色值的RGBA联合存储

## 问题描述
使用结构体和联合体实现RGBA颜色值的存储和操作。RGBA颜色值包含四个分量：红色(R)、绿色(G)、蓝色(B)和透明度(A)，每个分量都是一个0-255的整数。

## 要求
1. 定义RGBA颜色结构体，包含四个颜色分量
2. 使用联合体将RGBA结构体和32位整数值关联
3. 实现以下功能：
   - 创建颜色值
   - 获取各个颜色分量
   - 设置各个颜色分量
   - 打印颜色信息
   - 显示32位整数值

## 输入输出格式
### 输入
- 创建颜色时输入四个0-255的整数值，分别表示R、G、B、A分量
- 修改颜色分量时输入新的分量值

### 输出
- 显示颜色信息时，同时显示RGBA值和对应的32位整数值
- 格式示例：
  ```
  RGBA(255, 0, 0, 255)
  32位整数值: 0xFFFF0000
  ```

## 示例
```
颜色值RGBA联合存储程序
====================

颜色1 (红色):
RGBA(255, 0, 0, 255)
32位整数值: 0xFFFF0000

颜色2 (半透明绿色):
RGBA(0, 255, 0, 128)
32位整数值: 0x8000FF00
```

## 提示
1. 使用unsigned char类型存储颜色分量
2. 使用unsigned int类型存储32位整数值
3. 注意内存对齐和字节序
4. 可以使用位运算操作颜色分量
