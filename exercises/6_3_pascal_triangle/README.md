# 练习6.3：实现杨辉三角的存储与打印

## 题目描述
编写一个程序，生成并打印杨辉三角。杨辉三角是一个数字三角形，其中每个数字是上一行相邻两个数字的和。第一行只有一个数字1，第二行有两个数字1，以此类推。

## 要求
1. 实现一个函数`printPascalTriangle`，接收行数n作为参数
2. 使用二维数组存储杨辉三角
3. 按照等腰三角形的格式打印结果
4. 处理输入验证，确保n在合理范围内

## 输入格式
- 输入一个正整数n，表示杨辉三角的行数

## 输出格式
- 按照等腰三角形的格式打印杨辉三角
- 每个数字占4个字符宽度
- 使用空格对齐

## 示例
输入：
```
5
```
输出：
```
        1   
      1   1   
    1   2   1   
  1   3   3   1   
1   4   6   4   1   
```

## 提示
1. 使用二维数组存储杨辉三角
2. 每行的第一个和最后一个数字都是1
3. 其他数字是上一行相邻两个数字的和
4. 注意打印格式的对齐
5. 考虑使用动态内存分配来处理大行数
