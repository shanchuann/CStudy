# 练习4-10：超市收银系统

## 题目描述
编写一个程序，模拟超市收银系统。程序可以循环输入商品价格，输入0时结束，最后显示商品数量和总金额。

## 输入格式
- 循环输入商品价格（浮点数）
- 输入0表示结束

## 输出格式
- 显示欢迎信息
- 每次输入显示当前商品序号
- 最后显示收银小票，包含：
  - 商品数量
  - 商品总额（保留两位小数）

## 示例
```
欢迎使用超市收银系统！
请输入商品价格（输入0结束）：
商品1价格：12.5
商品2价格：8.8
商品3价格：15.2
商品4价格：0

=== 收银小票 ===
商品数量：3
商品总额：36.50
==============
```

## 提示
1. 使用float类型存储价格和总额
2. 使用int类型存储商品数量
3. 使用while(1)循环实现无限输入
4. 使用break语句在输入0时退出循环
5. 使用printf的格式控制符%.2f保留两位小数

## 学习要点
1. 浮点数处理
2. 循环控制
3. 格式化输出
4. 用户交互
