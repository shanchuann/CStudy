# 练习7.1：自定义strcpy函数

## 题目描述
编写一个程序，实现自定义的strcpy函数，用于将一个字符串复制到另一个字符串中。要求不使用标准库中的strcpy函数。

## 要求
1. 实现一个函数`my_strcpy`，接收目标字符串和源字符串作为参数
2. 正确处理字符串结束符'\0'
3. 返回目标字符串的起始地址
4. 处理输入验证
5. 不使用标准库中的strcpy函数

## 输入格式
- 输入一行字符串作为源字符串

## 输出格式
- 输出复制后的字符串

## 示例
输入：
```
Hello, World!
```
输出：
```
复制后的字符串: Hello, World!
```

## 提示
1. 使用指针操作实现字符串复制
2. 注意字符串结束符的处理
3. 考虑内存重叠的情况
4. 可以添加字符串长度检查
5. 考虑使用更安全的字符串操作函数
