# 数组第二大元素查找

## 题目描述
编写一个函数，在给定的整数数组中找出第二大的元素。如果不存在第二大的元素(例如所有元素都相同)，则返回-1。

## 函数要求
- 函数名：findSecondLargest
- 参数：
  - int arr[]: 整数数组
  - int size: 数组大小
- 返回值：第二大的元素，如果不存在则返回-1

## 输入格式
- 第一行输入数组大小n（1 ≤ n ≤ 100）
- 第二行输入n个整数，以空格分隔

## 输出格式
- 如果存在第二大元素，输出该元素
- 如果不存在，输出-1

## 示例
输入：
```
5
5 2 8 1 9
```
输出：
```
第二大的元素是：8
```
