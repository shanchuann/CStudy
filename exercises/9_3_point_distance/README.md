# 练习9.3：计算两点间距离

## 题目描述
编写一个程序，实现平面坐标系中点的相关计算。要求：
- 使用结构体表示点坐标
- 实现点间距离计算
- 实现点到原点距离计算
- 实现中点计算
- 实现三点共线判断

## 要求
1. 实现点结构
   - x坐标（double类型）
   - y坐标（double类型）
   - 创建和打印函数

2. 实现基本计算
   - 两点间距离
   - 点到原点距离
   - 两点间中点
   - 三点共线判断

3. 实现错误处理
   - 处理无效输入
   - 处理特殊情况
   - 处理精度问题

4. 实现用户界面
   - 显示功能菜单
   - 处理用户输入
   - 显示计算结果
   - 提供错误提示

## 输入格式
- 通过菜单选择操作
- 输入点坐标
- 输入计算参数
- 输入判断条件

## 输出格式
- 显示操作菜单
- 显示点坐标
- 显示计算结果
- 使用标准数学表示

## 示例
输出：
```
点坐标计算系统
1. 计算两点间距离
2. 计算点到原点距离
3. 计算两点间中点
4. 判断三点是否共线
0. 退出系统
请选择操作：1

请输入第一个点：
请输入x坐标：1
请输入y坐标：2
请输入第二个点：
请输入x坐标：3
请输入y坐标：4
点(1.00, 2.00)和点(3.00, 4.00)之间的距离为：2.83
```

## 提示
1. 使用结构体定义点
2. 实现基本计算函数
3. 注意精度控制
4. 处理特殊情况
5. 优化输出格式
