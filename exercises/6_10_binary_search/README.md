# 练习6.10：实现数组的二分查找

## 题目描述
编写一个程序，使用二分查找算法在已排序的数组中查找目标值。二分查找是一种高效的查找算法，通过不断将查找范围缩小一半来定位目标值。

## 要求
1. 实现一个函数`binarySearch`，接收数组、查找范围和目标值作为参数
2. 使用迭代方式实现二分查找
3. 处理目标值不存在的情况
4. 确保输入数组已排序
5. 处理输入验证

## 输入格式
- 第一行输入数组大小n
- 第二行输入n个已排序的整数
- 第三行输入要查找的目标值

## 输出格式
- 如果找到目标值，输出其在数组中的位置（从0开始）
- 如果未找到目标值，输出提示信息

## 示例
输入：
```
6
1 3 5 7 9 11
7
```
输出：
```
目标值7在数组中的位置是: 3
```

## 提示
1. 使用迭代而不是递归实现
2. 注意数组边界条件
3. 处理整数溢出
4. 考虑使用动态内存分配
5. 可以添加查找过程的输出
