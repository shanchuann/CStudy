学生成绩管理系统

编写一个程序，计算学生成绩的等级和绩点。
这个练习将帮助你理解条件判断、数据处理和格式化输出。

任务
编写一个程序，实现以下功能：
1. 从用户输入获取一个成绩值（0-100分）
2. 根据成绩计算等级（A/B/C/D/F）和绩点（4.0-0.0）
3. 输出计算结果
- 输入必须为有效成绩（0-100）
- 成绩等级和绩点对应关系：
  * 90-100: A (4.0)
  * 80-89:  B (3.0)
  * 70-79:  C (2.0)
  * 60-69:  D (1.0)
  * 0-59:   F (0.0)
- 输出格式为："等级,绩点" 或 "Invalid score"

学习要点
1. 条件判断和分支结构
2. 数据有效性检查
3. 格式化输出
4. 程序优化
