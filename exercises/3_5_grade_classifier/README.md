分数等级评价系统

编写一个程序，根据输入的分数判断其对应的等级。
在教育系统中，我们经常需要将百分制分数转换为等级制评价。这个练习将帮助你理解分数区间划分和等级判断的逻辑。

任务
编写一个程序，实现以下功能：
1. 从用户输入获取一个0-100之间的分数
2. 根据分数判断其对应的等级：
   - 90-100分：A
   - 80-89分：B
   - 70-79分：C
   - 60-69分：D
   - 0-59分：E
3. 输出对应的等级评价
- 输入分数必须是0-100之间的整数
- 如果输入超出范围，输出："输入分数无效"
- 输出格式为："您的分数是X分，等级为：Y"

学习要点
1. 区间判断
2. 条件分支结构
3. 输入验证
4. 格式化输出
