# 练习7.3：实现字符串的凯撒密码加密

## 题目描述
编写一个程序，实现凯撒密码加密。凯撒密码是一种替换加密技术，通过将字母表中的每个字母移动固定的位数来实现加密。例如，当移位值为3时，'A'变成'D'，'B'变成'E'，以此类推。

## 要求
1. 实现一个函数`caesarCipher`，接收字符串和移位值作为参数
2. 只对字母进行加密，保持其他字符不变
3. 分别处理大写字母和小写字母
4. 处理移位值超出范围的情况
5. 保持原始字符串的格式

## 输入格式
- 第一行输入要加密的文本
- 第二行输入移位值（1-25）

## 输出格式
- 输出加密后的文本

## 示例
输入：
```
Hello, World!
3
```
输出：
```
加密后的文本: Khoor, Zruog!
```

## 提示
1. 使用模运算处理字母循环
2. 使用isalpha和isupper函数判断字符类型
3. 注意处理大小写字母的基准值
4. 考虑使用更安全的输入函数
5. 可以添加解密功能
