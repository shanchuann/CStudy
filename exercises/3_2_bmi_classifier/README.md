# BMI肥胖等级判断

编写一个程序，根据用户的身高和体重计算BMI值，并判断其肥胖等级。

## 描述

BMI（身体质量指数）是衡量人体胖瘦程度的重要指标。通过计算BMI值，我们可以判断一个人的体重是否处于健康范围。这个练习将帮助你理解条件判断的实际应用。

## 任务

编写一个程序，实现以下功能：
1. 从用户输入获取身高（米）和体重（千克）
2. 计算BMI值（BMI = 体重/身高²）
3. 根据BMI值判断肥胖等级

BMI分类标准：
- BMI < 18.5：体重过轻
- 18.5 ≤ BMI < 24：体重正常
- 24 ≤ BMI < 28：体重过重
- BMI ≥ 28：肥胖

要求：
- 身高和体重使用浮点数表示
- BMI值保留一位小数
- 输出格式为："您的BMI值为：X，属于：Y"

## 学习要点

1. 浮点数计算和精度控制
2. 多重条件判断
3. 数学计算和公式实现
4. 结果分类和输出格式化
