# 练习6.8：将数组元素循环右移k位

## 题目描述
编写一个程序，将数组中的元素循环右移k位。例如，数组[1,2,3,4,5]右移2位后变成[4,5,1,2,3]。要求使用原地算法，不创建新的数组。

## 要求
1. 实现一个函数`rotateRight`，接收数组、数组大小和右移位数作为参数
2. 使用原地算法实现右移
3. 处理k大于数组大小的情况
4. 输出移动后的数组

## 输入格式
- 第一行输入数组大小n
- 第二行输入右移位数k
- 第三行输入n个整数

## 输出格式
- 输出右移k位后的数组，每个数字之间用空格分隔

## 示例
输入：
```
5
2
1 2 3 4 5
```
输出：
```
右移2位后的数组: 4 5 1 2 3
```

## 提示
1. 使用三次反转实现右移
2. 注意处理k大于n的情况
3. 考虑使用动态内存分配
4. 可以添加输入验证
5. 优化空间复杂度
