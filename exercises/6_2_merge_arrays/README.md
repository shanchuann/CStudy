# 练习6.2：合并两个有序数组

## 题目描述
编写一个程序，将两个有序数组合并成一个新的有序数组。要求保持数组的有序性，并且不使用额外的排序操作。

## 要求
1. 实现一个函数`mergeArrays`，接收两个有序数组和它们的大小，以及一个结果数组作为参数
2. 函数应将两个有序数组合并成一个新的有序数组
3. 主函数中实现两个数组的输入和结果的输出
4. 确保合并后的数组保持有序性

## 输入格式
- 第一行输入第一个数组的大小n1
- 第二行输入n1个有序整数
- 第三行输入第二个数组的大小n2
- 第四行输入n2个有序整数

## 输出格式
- 输出合并后的有序数组，每个数字之间用空格分隔

## 示例
输入：
```
3
1 3 5
4
2 4 6 8
```
输出：
```
合并后的数组: 1 2 3 4 5 6 8
```

## 提示
1. 使用双指针技术来合并数组
2. 注意处理数组长度不同的情况
3. 确保结果数组有足够的空间存储所有元素
4. 考虑使用const修饰输入参数
