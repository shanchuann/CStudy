自动售货机找零系统

自动售货机是日常生活中常见的设备，它需要能够准确计算找零金额。这个练习将帮助你理解货币计算和找零的逻辑。

任务
编写一个程序，实现以下功能：
1. 从用户输入获取商品价格（整数，单位：元）
2. 从用户输入获取支付金额（整数，单位：元）
3. 计算并输出找零金额
4. 输出找零的纸币和硬币组合
要求：
- 假设纸币面额有：100元、50元、20元、10元、5元
- 硬币面额有：2元、1元、5角、2角、1角
- 优先使用大面额货币
- 输出格式示例：
  找零：X元
  100元：X张
  50元：X张
  20元：X张
  10元：X张
  5元：X张
  2元：X个
  1元：X个
  5角：X个
  2角：X个
  1角：X个

学习要点
1. 整数除法和取余运算
2. 货币计算和找零逻辑
3. 多重条件判断
4. 格式化输出
