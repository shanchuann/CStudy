# 练习9.7：创建多项式相加的链表结构

## 题目描述
编写一个程序，使用链表结构实现多项式的相加操作。要求：
- 使用链表存储多项式
- 实现多项式的相加运算
- 按指数降序排列
- 合并同类项

## 要求
1. 实现链表结构
   - 定义多项式项结构体
   - 定义多项式结构体
   - 实现链表操作
   - 管理内存分配

2. 实现多项式操作
   - 添加多项式项
   - 合并同类项
   - 多项式相加
   - 按指数排序

3. 实现用户界面
   - 输入多项式
   - 显示多项式
   - 显示计算结果
   - 格式化输出

4. 实现内存管理
   - 动态分配内存
   - 释放内存
   - 处理内存错误
   - 防止内存泄漏

## 输入格式
- 输入多项式项数
- 输入每项的系数和指数
- 分别输入两个多项式

## 输出格式
- 显示输入的多项式
- 显示相加结果
- 使用标准多项式格式

## 示例
输入：
```
请输入第一个多项式：
请输入项数：3
请输入第1项的系数：2
请输入第1项的指数：3
请输入第2项的系数：-3
请输入第2项的指数：2
请输入第3项的系数：1
请输入第3项的指数：0

请输入第二个多项式：
请输入项数：2
请输入第1项的系数：4
请输入第1项的指数：3
请输入第2项的系数：2
请输入第2项的指数：1
```

输出：
```
多项式相加程序
==============

第一个多项式：2x^3 - 3x^2 + 1
第二个多项式：4x^3 + 2x

计算结果：
----------------------------------------
6x^3 - 3x^2 + 2x + 1
```

## 提示
1. 注意链表操作
2. 处理同类项合并
3. 按指数排序
4. 注意内存管理
5. 格式化输出结果
